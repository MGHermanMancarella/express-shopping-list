For this exercise we will be building a simple JSON API application where we will store a shopping list. You should use an array to store your items in the shopping list.

Each item should be a JavaScript object with the keys of name, and price.

Remember that since you are using an array for storage, this will be cleared each time the server restarts. Weâ€™ve given you a file, fakeDb.js, containing:

Your application should have the following routes:

GET /items: return list list of shopping items:

{ items: [
  { name: "popsicle", price: 1.45 },
  { name: "cheerios", price: 3.40 }
]}

POST /items: accept JSON body, add item, and return it:

{name: "popsicle", price: 1.45} =>
  {added: {name: "popsicle", price: 1.45}}

GET /items/:name: return single item:

{name: "popsicle", "price": 1.45}

PATCH /items/:name: accept JSON body, modify item, return it:

{name: "new popsicle", price: 2.45} =>
  {updated: {name: "new popsicle", price: 2.45}}

DELETE /items/:name: delete item:

{message: "Deleted"}

Please make use of the Express Router for all of your routes.
